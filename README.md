## features
* Rest & Soap consumer library
* Webservice caching
* Logging

## Installation

### Installation by Composer

If you use composer, add library as a dependency to the composer.json of your application

```php
    "require": {
        ...
        "itkg/consumer": "dev-master"
        ...
    },

```
